Operational drone-status pill — dot + mono caps. Use on fleet cards and rosters. Distinct from `ThreatBadge` (which is the perimeter condition).

```jsx
<StatusPill status="PATROLLING" />
<StatusPill status="CHARGING" />
```

Statuses: `PATROLLING` (teal), `READY` (slate-blue), `CHARGING` (amber), `RTN` (blue), `MAINTENANCE` (grey).
